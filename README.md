DevTrack

A full-stack web application that helps developers and students manage, organize, and track their development projects efficiently in one centralized dashboard.

📌 Overview

DevTrack allows users to create an account, log in securely, and manage their development projects.
It provides a structured way to track project progress, technologies used, and GitHub repositories.

The platform helps developers stay organized while working on multiple projects and improves productivity.

🎯 Problem Statement

Developers and students often work on multiple projects simultaneously, but they lack a centralized system to track:

Which projects are completed

Which are still in progress

Which are in planning

What technologies were used

Where the GitHub repositories are located

This leads to poor organization and productivity.

DevTrack solves this problem by providing a clean and interactive dashboard that organizes all projects in one place.

✨ Features
🔐 Authentication System

User Registration

User Login

JWT-based authentication

Protected routes

Secure logout functionality

Only authenticated users can access and manage their projects.

📂 Project Management

Users can create and manage their development projects.

Each project includes:

Project Title

Project Status

Planning

In Progress

Completed

Technology Stack

GitHub Repository Link

All projects are linked to the logged-in user.

📊 Dashboard & Statistics

The dashboard provides a quick overview of project progress.

It displays:

Total Projects

Completed Projects

Projects In Progress

Projects In Planning

Statistics are calculated dynamically from the database.
